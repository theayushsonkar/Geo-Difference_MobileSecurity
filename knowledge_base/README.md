# Privacy-Sensitive API Knowledge Base

This module serves as the authoritative source of truth for the Android Manifest Scanner. It consolidates multiple authoritative Android datasets (Axplorer, PScout, GMS) into a single, canonical, version-aware Privacy-Sensitive API Knowledge Base.

### Components
1. **Raw Datasets (`raw/`)**: Holds untouched, downloaded datasets (e.g., Axplorer text mappings).
2. **Importers (`importers/`)**: Contains parsers that extract and map API signatures strictly against the canonical taxonomy.
3. **Schemas (`schemas/`)**: Strict Python dataclasses (`PrivacyAPIRecord`, `PrivacyCategory`) defining the database structure.
4. **Metadata (`metadata/`)**: Tracks dataset provenance, categories, and official Android permission taxonomies.
5. **Processed (`processed/`)**: Destination for parsed, cleaned CSV outputs generated by the importers.
