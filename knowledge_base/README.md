# Knowledge Base & Detection Engine

This module serves as the authoritative detection intelligence layer for the Geo-Difference Mobile Security static analysis system. It is responsible for parsing disparate external security datasets, merging them deterministically, and executing high-performance runtime detection against Smali bytecode.

## Complete Technical Documentation

The authoritative, fully up-to-date documentation for this module — detailing every external dataset used (Axplorer, PScout, GMS, FlowDroid, TruffleHog), the exact deduplication and merge pipeline logic, and the high-performance runtime algorithms (Aho-Corasick, RE2 Regex) — is located at:

**[docs/OVERVIEW.md](docs/OVERVIEW.md)**

Please refer to `OVERVIEW.md` for any questions regarding architecture, data pipelines, schema layouts, or matching mechanisms.

## Directory Structure

*   **`pipeline/`**: The runtime detection engine components (`matcher_factory.py`, `aho_matcher.py`, `knowledge_enrichment.py`).
*   **`importers/`**: Build scripts to ingest and normalize raw external datasets.
*   **`metadata/`**: Immutable taxonomy tables, classification rules, and regex definitions.
*   **`processed/`**: The canonical, frozen outputs generated by the build pipeline (e.g., `privacy_apis.csv`).
*   **`schemas/`**: Strict Python dataclasses (`PrivacyAPIRecord`, `BaseFinding`) defining the system types.
*   **`docs/`**: Technical overviews, architectural references, and audit logs.
